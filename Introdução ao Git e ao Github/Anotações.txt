O GitBash é um terminal extendido para otimizar o uso do Git.

## Comandos úteis Git:
 - git status = Serve para visualizar o status da pasta aberta no terminal
 - git add * = Serve para adicionar todo o contéudo da pasta no git
 - git commit -m "" = Serve para criar um commit dessa mudança
 - git push origin main/master = Serve para enviar o arquivo para o GitHub
 - git clone + link = Serve para clonar um repositório do Github
